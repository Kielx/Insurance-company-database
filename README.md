<h3 align="center">Baza danych i hurtownia danych firmy ubezpieczeniowej</h3>

  <p align="center">
    Celem projektu było stworzenie fragmentu bazy danych firmy ubezpieczeniowej i opartej na niej hurtowni danych. Baza zawiera informacje o klientach, pracownikach i oddziałach firmy ubezpieczeniowej oraz o polisach ubezpieczeniowych zawieranych między tymi podmiotami. Baza i hurtownia zaprojektowana w ten sposób pozwala na łatwą wizualizację i opracowanie zestawień pracy wszystkich oddziałów, pracowników, a także polis zawieranych przez klientów. Założono, iż firma może oferować różnego rodzaju polisy –
np. ubezpieczenie domu, pojazdu – a każdy klient może zawierać nieograniczoną liczbę polis. Projekt został w całości przygotowany w środowisku oferowanym przez firmę Oracle, przy wykorzystaniu oprogramowania SQL Developer i SQL Developer Data Modeler, a także z wykorzystaniem narzędzia SQL*Loader. Skrypty generujące dane, którymi zasilono bazę napisane zostały w języku Python z wykorzystaniem biblioteki Faker.

<!-- GETTING STARTED -->
## Getting Started

By rozpocząć pracę z tym projektem musisz zainstalować bazę danych Oracle Express.
Instrukcje jak przeprowadzić ten proces znajdziesz tu:
[Oracle Database XE Quick Start](https://www.oracle.com/pl/database/technologies/appdev/xe/quickstart.html)

Bardziej doświadczeni mogą wykorzystać obraz Docker: 
[Oracle Container Registry](https://container-registry.oracle.com/) 
Należy szukać odpowiedniego obrazu w sekcji Database -> Express

Model bazy danych stworzony został w [Oracle SQL Data Modeler](https://www.oracle.com/pl/database/sqldeveloper/technologies/sql-data-modeler/). Narzędzie to przydaje sięby łatwiej wizualizować i tworzyć strukturę bazy.

Do obsługi bazy korzystałem z [SQL Developer](https://www.oracle.com/database/sqldeveloper/)

Kiedy mamy wszystkie narzędzia możemy przystąpić do uruchomienia projektu.
